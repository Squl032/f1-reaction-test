# 🏎️ F1 Reaction Test Pro (F1 起跑反應測試)

這是一個高擬真的 F1 起跑反應測試網頁應用程式。模擬真實的一級方程式賽車起跑燈號程序，結合精準的音效同步技術，讓你在電腦或手機上都能測試自己是否具備賽車手般的反應神經。

👉 **[點擊這裡開始遊玩 (Live Demo)](https://squl032.github.io/f1-reaction-test/)**

## ✨ 特色功能 (Features)

* **🚦 擬真燈號程序**：完全重現 F1 賽事標準的「五燈亮起 -> 熄滅起跑」流程。
* **🔊 專業音效同步**：
    * 支援自訂 `mp3` 起跑音效。 (僅本地運行)
    * 內建「智慧切片」技術，精準抓取音效節奏，與燈號完美同步。
    * **合成搶跑音效**：若在燈滅前操作，會觸發即時合成的錯誤音效 (Synthesized Fail Sound)。
* **⚙️ 進階音訊校準 (Audio Calibration)**：
    * 專為不同音源設計的校準面板。
    * 可調整 **起始延遲 (Offset)**：消除音檔前段空白。
    * 可調整 **播放長度 (Duration)**：控制嗶聲長短。
    * **自動記憶**：設定會自動儲存在瀏覽器中，下次遊玩無需重新設定。
* **📱 跨平台支援**：完美支援桌機、平板與手機（iOS/Android），針對觸控操作優化，無延遲。
* **⏱️ 毫秒級計時**：燈滅瞬間即時顯示計時器，精確捕捉你的反應時間。

## 🎮 如何遊玩 (How to Play)

1.  **開啟網頁**：點擊上方連結進入測試。
2.  **開始程序**：點擊螢幕任意處（或按下空白鍵/Enter）。
3.  **等待亮燈**：五顆紅燈會依序亮起（伴隨音效）。
4.  **全神貫注**：五燈全亮後，會隨機等待 0.2 ~ 3 秒。
5.  **起跑！**：當 **燈光熄滅** 的瞬間，立即點擊螢幕！
6.  **查看成績**：
    * 🏆 **< 200ms**: F1 車手等級！
    * 🐢 **> 500ms**: 該醒醒囉！
    * ❌ **Jump Start**: 搶跑（燈滅前偷跑），視為失敗。

## ⚙️ 音效校準說明 (Configuration)

若你發現亮燈與聲音不同步，可以點擊畫面 **左上角的齒輪圖示** 進入校準模式：

* **起始延遲 (Start Offset)**：調整跳過音檔前幾秒的空白。預設為 `0.64s`（針對標準 F1 音效優化）。
* **播放長度 (Duration)**：調整每次亮燈時「嗶」聲的持續時間。
* **試聽按鈕**：調整後可直接試聽，確保聲音乾淨俐落。

## 🛠️ 本地安裝與開發 (Installation)

如果你想在自己的電腦上運行或修改此專案：

1.  **複製專案**：
    ```bash
    git clone https://github.com/squl032/f1-reaction-test.git
    ```
2.  **準備音檔**：
    確保目錄下有名為 `f1-sound.mp3` 的音效檔案。
3.  **執行**：
    直接用瀏覽器開啟 `index.html` 即可。
